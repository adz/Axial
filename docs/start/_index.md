# Start

F# gives you excellent tools for modelling data, but production systems keep exposing the same gaps: validation that
either stops at the first error or buries you in applicative plumbing, invalid objects that get constructed before
anyone checks them, and infrastructure — connections, config, cancellation, telemetry — hand-threaded through every
function. Axial closes those gaps with a small set of tools that share one vocabulary, and the right move is almost
always the *smallest* one that fits.

Begin here to choose that smallest tool for the work in front of you.

- [Getting Started](./getting-started/): route between Result, Validation, and Flow.
- [Straightforward Examples](./basic-examples/): small examples in context.
