---
title: "PositiveInt"
---

`PositiveInt` functions construct, inspect, and transform positive integers.

- [`Refined.PositiveInt.value`](./m-refined-positiveint-value.md): Returns the underlying integer value.
- [`Refined.PositiveInt.create`](./m-refined-positiveint-create.md): Builds a positive integer.
- [`Refined.PositiveInt.map`](./m-refined-positiveint-map.md): Transforms the value and re-certifies the positive integer invariant.
- [`Refined.PositiveInt.replace`](./m-refined-positiveint-replace.md): Replaces the value and re-certifies the positive integer invariant.
