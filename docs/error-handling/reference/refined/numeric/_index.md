- [`Refined.Numeric.negativeInt`](./m-refined-numeric-negativeint.md): Builds a negative integer.
- [`Refined.Numeric.nonPositiveInt`](./m-refined-numeric-nonpositiveint.md): Builds a non-positive integer.
