---
title: "Temporal"
---

`Temporal` functions construct refined date and time values.

- [`Refined.Temporal.dateTimeOffsetRange`](./m-refined-temporal-datetimeoffsetrange.md): Builds a date and time range where <code>Start &lt;= End</code>.
- [`Refined.Temporal.dateOnlyRange`](./m-refined-temporal-dateonlyrange.md): Builds a date-only range where <code>Start &lt;= End</code>.
