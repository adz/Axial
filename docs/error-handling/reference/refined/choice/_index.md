---
title: "Choice"
---

This page shows fallback helpers for alternative refinements.

- [`Refined.Choice.orElse`](./m-refined-choice-orelse.md): Tries the left parser first, then the right parser, mapping either success into a caller-owned output type.
- [`Refined.Choice.tryAny`](./m-refined-choice-tryany.md): Tries parser strategies in order and returns the first success.
