---
title: "Character"
---

This page shows character predicates used by refinements.

- [`Refined.Character.isAsciiDigit`](./m-refined-character-isasciidigit.md): Returns true when the character is an ASCII digit.
- [`Refined.Character.isAsciiHexDigit`](./m-refined-character-isasciihexdigit.md): Returns true when the character is an ASCII hexadecimal digit.
- [`Refined.Character.isLowercase`](./m-refined-character-islowercase.md): Returns true when the character is lowercase according to invariant Unicode casing.
- [`Refined.Character.isUppercase`](./m-refined-character-isuppercase.md): Returns true when the character is uppercase according to invariant Unicode casing.
- [`Refined.Character.isWhitespace`](./m-refined-character-iswhitespace.md): Returns true when the character is whitespace.
- [`Refined.Character.isControl`](./m-refined-character-iscontrol.md): Returns true when the character is a control character.
- [`Refined.Character.isNumeric`](./m-refined-character-isnumeric.md): Returns true when the character is numeric according to Unicode character data.
