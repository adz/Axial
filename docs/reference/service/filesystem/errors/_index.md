This page shows the `FileSystemError` helpers for classifying and describing file-system failures.

- [`Flow.FileSystem.FileSystemError.fromException`](./m-flow-filesystem-filesystemerror-fromexception.md): Classifies an exception raised by a file-system operation.
- [`Flow.FileSystem.FileSystemError.describe`](./m-flow-filesystem-filesystemerror-describe.md): Formats a human-readable description for a file-system error.
