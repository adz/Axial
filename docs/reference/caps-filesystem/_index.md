This page shows the source-documented `FsFlow.Caps.FileSystem` surface: the file system interface and its helpers.

## Capability

- [`FsFlow.Caps.FileSystem.IFileSystem`](./t-filesystem-ifilesystem.md): Provides synchronous access to file system operations.

## Helpers

- [`FsFlow.Caps.FileSystem.FileSystemModule.readAllText`](./m-filesystem-filesystemmodule-readalltext.md): Reads all text from a file using the file system environment.
- [`FsFlow.Caps.FileSystem.FileSystemModule.writeAllText`](./m-filesystem-filesystemmodule-writealltext.md): Writes all text to a file using the file system environment.
- [`FsFlow.Caps.FileSystem.FileSystemModule.exists`](./m-filesystem-filesystemmodule-exists.md): Checks if a file exists using the file system environment.
- [`FsFlow.Caps.FileSystem.FileSystemModule.live`](./p-filesystem-filesystemmodule-live.md): Creates a live file system backed by `File`.
