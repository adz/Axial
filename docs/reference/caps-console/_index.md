This page shows the source-documented `FsFlow.Caps.Console` surface: the console interface and its helpers.

## Capability

- [`FsFlow.Caps.Console.IConsole`](./t-console-iconsole.md): Provides synchronous access to standard console I/O.

## Helpers

- [`FsFlow.Caps.Console.ConsoleModule.readLine`](./m-console-consolemodule-readline.md): Reads a line from the console environment.
- [`FsFlow.Caps.Console.ConsoleModule.writeLine`](./m-console-consolemodule-writeline.md): Writes a line to the console environment.
- [`FsFlow.Caps.Console.ConsoleModule.live`](./p-console-consolemodule-live.md): Creates a live console backed by `Console`.
