This page shows the `FileSystem.FileSystem` helpers for reading and updating file metadata.

- [`Flow.FileSystem.getFileAttributes`](./m-flow-filesystem-filesystem-getfileattributes.md): Gets file attributes through an explicit file-system service.
- [`Flow.FileSystem.setFileAttributes`](./m-flow-filesystem-filesystem-setfileattributes.md): Sets file attributes through an explicit file-system service.
- [`Flow.FileSystem.getFileCreationTime`](./m-flow-filesystem-filesystem-getfilecreationtime.md): Gets file creation time through an explicit file-system service.
- [`Flow.FileSystem.getFileCreationTimeUtc`](./m-flow-filesystem-filesystem-getfilecreationtimeutc.md): Gets file creation time in UTC through an explicit file-system service.
- [`Flow.FileSystem.setFileCreationTime`](./m-flow-filesystem-filesystem-setfilecreationtime.md): Sets file creation time through an explicit file-system service.
- [`Flow.FileSystem.setFileCreationTimeUtc`](./m-flow-filesystem-filesystem-setfilecreationtimeutc.md): Sets file creation time in UTC through an explicit file-system service.
- [`Flow.FileSystem.getFileLastAccessTime`](./m-flow-filesystem-filesystem-getfilelastaccesstime.md): Gets file last access time through an explicit file-system service.
- [`Flow.FileSystem.getFileLastAccessTimeUtc`](./m-flow-filesystem-filesystem-getfilelastaccesstimeutc.md): Gets file last access time in UTC through an explicit file-system service.
- [`Flow.FileSystem.setFileLastAccessTime`](./m-flow-filesystem-filesystem-setfilelastaccesstime.md): Sets file last access time through an explicit file-system service.
- [`Flow.FileSystem.setFileLastAccessTimeUtc`](./m-flow-filesystem-filesystem-setfilelastaccesstimeutc.md): Sets file last access time in UTC through an explicit file-system service.
- [`Flow.FileSystem.getFileLastWriteTime`](./m-flow-filesystem-filesystem-getfilelastwritetime.md): Gets file last write time through an explicit file-system service.
- [`Flow.FileSystem.getFileLastWriteTimeUtc`](./m-flow-filesystem-filesystem-getfilelastwritetimeutc.md): Gets file last write time in UTC through an explicit file-system service.
- [`Flow.FileSystem.setFileLastWriteTime`](./m-flow-filesystem-filesystem-setfilelastwritetime.md): Sets file last write time through an explicit file-system service.
- [`Flow.FileSystem.setFileLastWriteTimeUtc`](./m-flow-filesystem-filesystem-setfilelastwritetimeutc.md): Sets file last write time in UTC through an explicit file-system service.
