<code><span>HostContext.withHost&#32;<span>host&#32;context</span></span></code>

Replaces the host half of a host context.

## Parameters

- `host`: <code>'nextHost</code>
  The new host services.
- `context`: <code><span><a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-hostcontext-2.html">HostContext</a>&lt;<span>'host,&#32;'appEnv</span>&gt;</span></code>
  The source context.

## Returns

A new context with the replaced host services.
