This page shows the core `Schema<'model>` type. Schemas describe trusted model structure for interpreters such as input parsing, validation, codecs, JSON Schema, UI, and documentation. The core schema package stays independent of workflow execution, diagnostics, raw input, and validation interpreters.

## Core type

- [`Schema.Schema`](./t-schema-schema.md):
 Describes the portable structure of a trusted model for schema interpreters.
