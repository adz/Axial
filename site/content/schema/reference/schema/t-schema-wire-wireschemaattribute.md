Marks a plain record as a wire DTO whose permissive schema is generated by <code>schemagen</code>.
 The attributes in this namespace are inert metadata: they are read from source text at generation time,
 never by runtime reflection, and their vocabulary mirrors the <code>.contract</code> constraint grammar
 one-to-one.
